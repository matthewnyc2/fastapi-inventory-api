# Orders Management Console Screen

## Overall Vibe
Order tracking and management interface. Clean, data-focused layout with order status filters and action buttons.

## Design System
- Order table with filterable status badges
- Expandable order details
- Action buttons for status transitions
- Timeline showing order progression

## Page Structure
Header: "Orders Management"
Status filter buttons: All, Pending, Confirmed, Processing, Shipped, Completed, Cancelled
Orders table with columns: Order ID, Customer, Items Count, Status, Total Amount, Date, Actions
Sample orders with mixed statuses
Expandable detail view for selected order showing line items
