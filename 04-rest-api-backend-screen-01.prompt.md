# API Documentation Console Screen

## Overall Vibe
Professional API documentation interface with dark theme. Clean, organized endpoint explorer with syntax-highlighted request/response examples.

## Design System
- Dark background with light text
- Syntax highlighting for code
- Organized endpoint categories (Auth, Products, Orders, Inventory)
- Interactive request builder

## Page Structure
Left sidebar: API endpoint categories and methods
Main content: Selected endpoint documentation showing:
- Endpoint path and HTTP method
- Description
- Request parameters table
- Response example (JSON with syntax highlighting)
- Try It button for testing
